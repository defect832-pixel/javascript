Script Placement

If you want to connect JavaScript to an HTML page, you use the script tag. Placing a script tag in the head can sometimes cause errors if the script tries to Access Page elements that haven't been created yet, because the browser reads HTML top to bottom.

For example, if you have an H1 Inside the body, and your script, in the head, tries to change that H1 before it exists, the script will fail.

Solution 1: Put the script tag as the last line inside the body.
Solution 2: Use the defer attribute (Recommended).

index.html
1<!-- Modern Safe Loading -->
2<head>
3  <script src="main.js" defer></script>
4</head>

System Output

console.log()

console.log() is a JavaScript method used to print messages to the browser’s console. Developers use it to check values, track the flow of the code, and understand what’s happening while the program runs. It doesn’t stop the execution of the code and doesn’t appear to the user — it’s only visible in the developer tools. This makes it perfect for debugging, testing ideas, and displaying helpful information during development.

main.js
1 // إرسال رسالة إلى سجلات النظام
2 console.log("System Ready");
console.error()

console.error() works like console.log(), but it is specifically used to show error messages in the console. The browser highlights these messages differently (usually in red) to make them stand out, helping developers quickly notice problems in the code. It doesn’t interrupt the program, but it provides a clearer way to track issues, warnings, or anything that should be treated as an error during development.

error_handler.js
1 console.error("Connection Lost!");

User Interface Interaction

window.alert()

A built-in JavaScript message used to display a simple message in a dialog box.

When an alert appears, it stops execution and blocks the entire page until OK is clicked. Not ideal for real UI, as you cannot style it with CSS. It is mainly used for testing and learning.

alert.js
1window.alert("this is alert");
document.write()

Injects raw data directly into the HTML stream.

Warning: Executing this command after the page has fully loaded will trigger a document.open(), effectively wiping the entire DOM and destroying the interface.

write.js
1document.write("<h1>Hi</h1>");
window.close()

A method used to close the current browser window. However, modern browsers only allow a script to close a window if that window was organically opened by JavaScript (using window.open).

If the user opened the page normally, this function does nothing for security reasons.

close.js
1window.close();
